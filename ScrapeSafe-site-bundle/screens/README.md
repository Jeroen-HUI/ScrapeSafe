# Carousel screenshots (marketing)

The hero carousel uses **10 PNGs** with stable names:

| File | Source creative (from your exports) |
|------|-------------------------------------|
| `slide-01.png` | Map / Amsterdam-style hazard view |
| `slide-02.png` | Add hazard |
| `slide-03.png` | Feed / share your car |
| `slide-04.png` | Groups |
| `slide-05.png` | Join or host meets |
| `slide-06.png` | Group-only meet setup |
| `slide-07.png` | Garage / add car |
| `slide-08.png` | Can I clear it |
| `slide-09.png` | English / Dutch settings |
| `slide-10.png` | Map tab “for people who drive low” |

To replace an image, keep the same filename so `index.html` stays unchanged. Prefer WebP later with `<picture>` and `srcset` for performance.

Extra assets in your library (app icon, wordmark banner) can live under `/brand/` if you add them to the layout separately.
